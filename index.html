Major Map Initiative

The prerequisite mapper lists all the prerequisites for a given class in an intuitive, interactive diagram.

Check out the tutorial below the prerequisite mapper to understand how to use it.

Tutorial

Hover on a course node to display the title
Click on a course node to show its description
Maps may generate messily: drag nodes around to rearrange however you want

Arrow colors indicate groupings.
-Arrows of the same color mean that either one satistfies
the course they are pointing to.
For example COGS 109 or COGS 108 (green arrows) can
be taken before COGS 118A.

- Black arrows indicate that there are no
alternatives for that prerequisite.
Nodes labelled 'other' may include extra details
and prerequisites
